# Employee Test Application
Diese Anwendung ist ein Übungsspielplatz, der in der Freizeit entsteht. Angular im FE, Node.js im BE und bislang MongoDB als einzige Datenbank. Ziel ist es, verschiedene Funktionen einer Mitarbeiteranwendung zu implementieren. Die erste Funktion ist die Mitarbeiterverwaltung,
in der Mitarbeiter angelegt, eingesehen, bearbeitet und gelöscht werden können. Künftig soll es noch mehr Unterfunktionen hier geben, wie Filtern, randomisiertes Anlegen mehrerer Mitarbeiter usw. Die Anwendung entsteht nur aus Spaß am Programmieren und dient eben nur zu
Übungszwecken. Neben der Verwaltungsfunktion ist der Plan, irgendwann eine Art internes Social Media zwischen den fiktiven Mitarbeitern zu kreieren, vermutlich mit Neo4j als Graphdatenbank im Hintergrund.
